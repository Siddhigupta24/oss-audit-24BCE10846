📘 The Open Source Audit — LibreOffice
📌 Overview

This project is a comprehensive audit of LibreOffice, a widely used open-source office productivity suite. It analyzes the software from multiple perspectives including its origin, licensing, ethical implications, Linux integration, ecosystem, and comparison with proprietary alternatives.

The report is part of the OSS NGMC Course Capstone Project.
